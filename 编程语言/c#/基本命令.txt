1.基本操作：
	>dotnet new console -o 路径 ：初始化一个项目
	  dotnet run 项目所在路径下运行。则从Main方法开始执行。
	  dotnet run -p cmdGenerate.csproj 项目根路径下执行，效果同上。
	  dotnet run  可以直接运行。在项目下即可。
        >dotnet publish cmdGenerate.csproj 发布工程。


	>环境中：F5可以快速执行工程。
		>快速运行ctrl + f5。
		>快捷键若干：https://blog.csdn.net/qq_32521313/article/details/52459573


 	>安装好.net framework：即在C:\Windows\System32里面有mscoree.dll文件。
	 >安装：下载.net sdk安装好即可，好的检测方法2：cmd中输入dotnet有输出。
	 >安装新lib: 下载dll文件，放到项目里Debug里，然后在解决方案右键导入项目引用。会在配置文件里增加：
		<ItemGroup>
    			<Reference Include="Newtonsoft.Json">
      			<HintPath>bin\Debug\lib\json\Newtonsoft.Json.dll</HintPath>
    			</Reference>
  		</ItemGroup>	

	>编译：
	  >生成：.exe文件 .dll动态库  .netmodule代码模块。。
	   >C:\Windows\Microsoft.NET\Framework64\v4.0.30319\csc.exe Program.cs com\file_handle\file1.cs com\Emp.cs -main:Test1.com.file_handle.file1
	   >对一个C#工程，指定一个Main方法入口类，生成一个exe文件：C:\Windows\Microsoft.NET\Framework64\v4.0.30319\csc.exe /out:firstOne.exe /t:exe /r:MSCorLib.dll  Program.cs com\file_handle\file1.cs com\Emp.cs -main:Test1.com.file_handle.file1
