1.问题：
	>命令：adb devices查看连接调试的手机:设备编号
		>安装apk开发包到手机：adb -s 设备编号 install test.apk
		>进入手机文件目录：adb shell       ls 大量的linux命令可以用
		>手机文件到电脑：adb pull /mnt/sdcard/Pictures/Screenshots/Screenshot_2018-03-07-21-58-57.png D:\abc.png
		>电脑文件到手机：adb push D:\test.png /mnt/sdcard/Pictures/Screenshots/abc.png
		
		>手机运行日志：adb logcat
		>无线连接pc:用同一个局域网ip链接的方式:adb connect 
		>操控手机-模拟按键：adb shell input keyevent 26  //按电源键
				    adb shell input keyevent 208 //打开计算器
				    adb shell input text 45 //输入文本框内容
				    adb shell screencap -p /sdcard/sc.png  //截屏图片 。后买可以adb pull /sdcard/sc.png D:\a.png转移到电脑上。
					//屏幕录制：screenrecord **.mp4
		
		>资源占用：adb shell top
		>进程查看：adb shell ps
