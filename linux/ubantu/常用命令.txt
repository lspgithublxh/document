2.权限更改：sudo chmod 777 filename 对这个文件设置当前用户、当前用户组、其他用户的权限。

                  ls -al >> 文件名.txt。追加方式。

11.磁盘和内存管理
   mount 挂载
   cfdisk 查看所有磁盘分区信息

12网络和硬件管理
  ifconfig
  ficonfig 网卡 up|down 激活网卡

  ip link show  显示网卡配置
  ping ip地址 向目标地址发送ICMP封包

28.改变文件的所有者或和群组：
  chown -R mysql data （只改变所有者）
  chown redhat3:redhat3 /home/test/ （改变了所有者和归的群组）
  chgrp -R mysql .  （递归改变当前所有文件所有组为mysql）
  chkconfig --add mysql.server  （增加一项服务）
