1.问题：
	>web网站开发：任何语言、web框架必须解决三个问题：
		>a.url-函数/资源的匹配(函数/方法级别)   
			>同时url上数据的获取-截取{}
		>b.数据库的访问(获取数据，序列化) 
		>c.后端数据填充到模板(html + velocity)(非jsp时)。以及多种数据的返回：直接json,txt,html,图片。。。动态静态资源

		>d.request,response,session三大实体必须提供。fiter,incepter也应当可以提供。

	>django: 
		>安装和新建工程：pip install django   ；django-admin startproject projectname
		>启动项目：进入项目：python manage.py runserver 0.0.0.0:8088
		>视图模板：
			>条件判断{% if condition %} {% elif %} {% else %} {% endif %}
			>循环{% for a in alist %}
			>值比较： ifequal
			>包含其他html: include 'nav.html'
		>数据库对象模型：
			>
 	>django网站特点：
		>url和视图改变不用重启服务器，自动可以生效。但是view处理器改变了需要重启。服务器自动监测代码并自动重新载入，无需手动启动。
	>pyspark:
		>启动spark集群，在spark目录下执行：./bin/run-example SparkPi
			>执行：./bin/spark-submit --master local[4] SimpleApp.py
			     ./bin/spark-submit examples/src/main/python/pi.py
				./bin/spark-submit examples/src/main/r/dataframe.R
