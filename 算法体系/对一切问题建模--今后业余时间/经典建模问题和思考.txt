					>泰勒级数的自然推导过程：在导数定义之后，x + Δx处的函数值f(x + Δx) 可以近似表达为：f'(x) * Δx + f(x) ,显然，当Δx增大之后，这个误差是增大的。而增大的原因在于x到x +Δx过程中，f'(x)的值发生了变化，所以，为了减小误差，精准的Δy 计算应该是：将Δx再分割为n份，每一份乘以其所在处的导数值，然后求和：f'(x)Δxn + f'(x + Δxn)Δxn + f'(x + 2Δxn)Δxn + ..... 而显然，从第二项开始，后一项可以用前一项表示出来，因为 f'(x + Δxn) - f'(x) = f''(x)Δxn, 从而第一轮替换，就可以得出用一阶和二阶导数表示的精确表达式，而且因为nΔxn = Δx, 再估计f''(x + kΔxn) = f''(x) ，和n(n - 1) Δxn2 = Δx2 那么可以得到一阶和二阶来表示的较精确表达式：Δy = f'(x) Δxn + 1/2 * f''(x)Δxn  。但是这个也是不很精确的，因为对f''(x + kΔxn) 做了简化估计，而实际上应该再次精确表示，即用导数定义而f''(x + Δxn) - f''(x) = f'''(x)Δxn 那么同理可以得出用三阶表示的精细表达式，而在同样估计式的计算过程中，发现f'''(x)的系数是n(n -1) + (n - 1)(n - 2) + ...这样的连续求和，完全是一个数论问题，而通过转化为自然数求和问题--和1 + 3 + 5 = 3^2这样的自然数平方和和自然数求和的关系，和转化为面积和问题，可以得出平方和公式: (n - 1)(n - 2) + (n - 2)(n - 3) + ... = 1/3 *(n - 2)(n - 1)(n),
							         从而得出用1-3阶导数表示的较精确表达式：Δy = f'(x)Δxn +1/2 * f''(x)Δxn^2 +1/(2 * 3) * f'''(x)Δxn^3  看到这个形式之后，以及想到估计时做的动作，如果用4阶乃至n阶导数来表示时，形式应该是很自然的推广：即Δy =  f'(x)Δxn +1/2 * f''(x)Δxn^2 +1/(2 * 3) * f'''(x)Δxn^3  + 1/(2 * 3 * 4) * f''''(x)Δxn^4 + ... + 1/n! * f'''(x)Δxn^n    而且估计到4阶进行的计算中，还自然推论出数论公式： (n - 1)(n - 2) * ..*(n - k) + (n - 2) * ..*(n - k - 1) + k * (k - 1) * 2 * 1 = 1/ (k + 1) n * (n - 1)(n - 2)(n - 3)...(n - k + 1)  。。。而显然的是，给定一个Δx，无论多大，只要 n足够大时--到足够高阶时，都能保证 Δy的值是正确的。所以，如果x的值取0，而Δx取任意x，那么就得出了y = Δy = f'(x)Δxn +1/2 * f''(x)Δxn^2 +1/(2 * 3) * f'''(x)Δxn^3  看到这个形式之后，以及想到估计时做的动作，如果用4阶乃至n阶导数来表示时，形式应该是很自然的推广：即y = f(0) + f'(x)x +1/2 * f''(x)x^2 +1/(2 * 3) * f'''(x)x^3  + 1/(2 * 3 * 4) * f''''(x)x^4 + ... + 1/n! * f'''(x)x^n   n --> +∞ 这就是泰勒级数。

					>傅立叶级数的自然推导过程：在研究振动中，知道基本的振动是三角函数形式，正弦余弦形式，只是振幅、相位、角速度不同，而从光的干涉、光的分解中知道单色光、同频率不同相位-振幅的光波可以合成-叠加增强或者减弱，而不同频率的光波又可以合成叠加而形成自然光，这种合成结果千变万化，自然令人想确定这种变化--即提出这种合成的一切可能结果是什么？不同振幅、角速度、相位的正弦振动若干个会合成什么振动？是否是任意的振动？是否任意的振动都能用不同振幅、频率、相位的正弦振动来模拟？并且通过调节和微调而不断减小误差直至无限接近某种一般的振动？比如0-1振动，周期为T,通过具体合成计算甚至都发现，通过将周期为T的正弦振动和更高频nT的多个正弦振动合成，确实发现非常接近0-1振动，且误差越来越小，可以猜测当n更大时，这种合成振动会无限接近0-1振动。从这个例子中，开始启发，试图推广，提出（问题和假设）：是否任意周期为T的函数都能用若干不同振幅、频率(必然是n倍,n为整数)、相位的正弦函数来合成来模拟并且让误差无限接近于0？假设为是。那么开始表达这个猜测：定量描述：周期函数f(t) 周期为T, 那么假设可以精确表达为：f(t) = a0 +∑ An sin( ｎωt +φｎ)　ｎ：1--＞+∞  自然试图简化，减少参数，而想到欧拉公式 sinx = (e^ix - e^-ix) / 2i，且转化过程中发现指数函数两项仅仅是x的取值不同，函数本身一样，从而可以用一个函数来表达---仅仅只需要拓展定义域，而转化为更简单的参数更少的表达式：f(t) = a0 + ∑ Xn e^(jｎωt)　ｎ：-∞--＞+∞ n≠0
								Xn的求解，可以从e^(jｎωt)函数本身性质中找，消除更高频的项是关键，现在知道的性质是, 一个周期T内：∫ e^(jｎωt) = 0 始终成立，而为了将Xk和指数函数分开，显然可以对f(t)乘以一个 e^(jkωt), 而其他项乘以 e^(-jkωt)后仍然保持指数形式，再利用之前的该指数函数积分为0性质，对剩余项在T内积分，自然就消除了，而Xk项不会消除。即∫ e^(-jkωt) * f(t) * dt = ∫ Xk dt = T * Xk 那么有 Xk = 1/T * ∫ e^(-jkωt) * f(t) * dt  ; k : -∞ --> +∞, k≠0 那么就只剩下a0的求解了，很明显一个周期T内： ∫ f(t) dt = ∫ a0 dt = T * a0 从而 a0 = 1/T * ∫ f(t) dt  从而有: f(t) = 1/T * ∫ f(t) dt + 1/T * ∑∫ e^(-jkωt) * f(t) * dt * e^(jnωt)  ；ｎ：-∞--＞+∞ n≠0 这就是著名的傅立叶级数的指数形式。
							>傅立叶变换的自然引出：上述求解Xn中，n取的每个值都要计算，比较复杂，而三角级数形式又和积分表达式的差分形式类似，而且三角级数形式表示中，其实可以看作是一个积分求面积的过程，把n当作自变量，那么相当于Δn = 1, 而显然当如果假设频率是连续的---即允许是小数--只要单位上放大其实可以近似是小数，那么这个傅立叶级数的连续形式就是：f(t) = ∫ X(ξ) * e^(jξt) dξ  这样，ξ把频率也包括进去了。那么此时自然关系的是X(ξ) 这个振幅随ξ的变化规律，即当频率在ξ时的振幅X(ξ)是多少。。而类似上述积分方式：这里也可以积分方式：X(η) = ∫ e^(-jηt) * f(t) dt 这个就是傅立叶变换。其中系数和一般的有区别，但是整体不影响。

			>变分问题：一般模式：求使得一个定积分取得最值的积分函数应该是什么样子。。其中，这个定积分中，被积函数是一个f(x) 和 它的各阶导数等和它相关的衍生的量的表达式。而这个由泛指函数构成的被积表达式构成的定积分 称作 泛函，，被积函数叫做 核。（泛指函数构成的函数）（区别于定函数在于：定函数的表达式的构成元素都是确定形态的基本函数和常量和基本运算构成的）(指数函数、三角函数可以当作时基本运算)
				>特点1：输入函数定，定积分的值确定；输入函数可以任意函数。
				>特点2：函数的微分是自变量临近微小变化时函数值的变化量；那么在泛函中，被积表达式中的独立函数的临近微小变化而导致定积分值的微小变化量叫做泛函的变分。
					>问题：泛指函数的临近变化是怎样的另一个泛指函数？泛指函数是一种运算(基本运算、基本运算组合构成的运算，新的非基本运算的新定义的运算---一种新映射)，泛指函数可以形象化为某区间上的任一条光滑曲线，泛指函数的集合就是某区间上所有光滑曲线的集合。泛指函数的集合称为赋范线性空间， 值的集合叫做数空间。
						>泛函的一般形式： J = ∫F(x, y, y' ) dx   而其中y = y(x)  。但是同时，F本身却可以看作是F(x, y, z)即三维直角坐标系下的一个函数，只是,z和y都是x的函数都相关而已。从而变分：F(x, y + δy, y' + δy') 看作是F(x, y, z)附近的一个值，而这个差值正好可以用F对y,z的偏微分表示，即δF = зF / зy * δy + зF / зy' * δy', 所以变分δJ = ∫[зF / зy * δy + зF / зy' * δy']dx  。。导数= 0是函数极值的必要条件，而导数=0的本质是dy =0 即随着x的变化，dx处的dy无限接近于0，那么就取得极值了，同样的，变分=0也是泛函取得极值的条件。因为δy也是x的函数--且x = a, x = b处值为0（因为δy是一个差函数，而任意两条曲线都起于a终于b，所以差值在a,b处都为0），所以用分布积分整理得：δJ = ∫[зF / зy * δy + зF / зy' * δy']dx = ∫[зF / зy  - d (зF / зy') / dx ]δydx   很显然，δJ存在始终为0的情形--不管δy如何变化--即不管其他任何函数和某个函数有多大差别，只要这个函数满足这种性质зF / зy  - d (зF / зy') / dx = 0， 那么泛函的变分始终为0，即δJ = 0 恒成立。那么这个函数就是使得泛函取得极值的函数。这个等式就是E-L方程。如果把E-L表达式当作是对某个泛指函数对x求导的化简结果，那么这个泛指函数可以是：F(y, y') - y' * зF(y, y') / зy'    所以F(y, y') - y' * зF(y, y') / зy'  = C也是泛函取得极值的必要条件的一种约束条件形式。

						>排列组合：组合的实际意义：存在组合因为实际可能结果就是组合的结果。而要使用组合的原因 是 组合 可以带来更多的结果---而一个组合结果可以表达一个量完全表达不出来的一个量无法确定的一种东西/情形/存在。多个变量可以确定一个向量，也可以确定空间中的一个点。矩阵，度量的是多个过程；比如 多个线性组合过程-----每个线性组合过程一行，每一行每个位置代表 参与线性组合 的 放在该位置上的变量/度量  的权重/系数 。。从行的角度看是前者，从列的角度看：则是同一个 度量/变量 在多个 线性组合 中 的 参与度/权重/系数。。。从结果看，一个平面点是一个组合结果，一个空间向量是一个组合结果，一个线性表达式是一个组合结果，。。。多轮组合式描述  会 真正彻底 描述 表达出 一组 变量的 性质/约束/ 各自的演变规律。所以矩阵的本质 是对一组变量进行 的 多轮 组合式描述(线性组合式描述)。而如果规定 每轮 组合 的结果 等于 /是 某个值，那么就是对这组变量之间互相依赖情况的一次描述，一次组合的 结果会是什么 的描述，；如果知道了n次组合的结果是什么 其实就基本确定了 这组变量 在 这n次组合 中 各自的 体现值 是什么。如果只是想要约束关系，那么这n个约束关系可以得出更少个变量的 组合的约束关系。最好的，是组合 中的 每个变量 用 n个 组合结果 + n次组合 的 每个变量的 参与度/权重/系数  来表出  。。

						>行列式：[C A; B 0] 的行列式，按照B从低阶到高阶展开来归纳，会发现展开式子的乘法因子组合 和 元素余子式 整体之间的加法组合 可以用一个三角形来描述，底边单位长度的点上是B的最底边的三个元素，依次往上是B的上一层元素---且变量随着底边不同位置元素而不同---但是符号却是惊人的统一，从而整个三角形左下顶角位置的元素的余子式的符号是(-1)^(2n+1), 向右依次加1,直到2n+n，而向上则依次减1，直到n+2；而最顶上的元素则直接 乘以A的行列式|A|， 这就是整个分块矩阵的行列式的展开形式的三角形组合形式，从而发现可以提出|A|出来，剩下的非常相似|B|的行列式，差别只在|B|的最底边元素的符号是(-1)^(n+1)，而符号的变化和因子组合都没有区别，所以符号相差n个-1，所以只要 三角形上底边每一项提出一个(-1)^(n)出来，剩下的表达式就构成了B的行列式，从而|[C A;B 0]| = (-1)^n|A||B|
							[A 0;C B]的行列式，只要按照A的第一行展开，就容易发现，也是一个三角形式组合来表达的行列式，但是左下角顶点的符号却固定是(-1)^2 所以 和|A|的行列式一致的，从而有：|[A 0;C B]| = |A||B|

						>继续定义：将样本空间分割为n份，每份称作一个子空间，那么任意一次实验结果的度量值落在 所有子空间构成的空间的概率是1，落在部分子空间的概率则不是1；假设Ω上有一个新的区间A,那么A和所有子空间的交集构成的空间就是∑A∩Bn = A∩Ω = A 所以P(A) =  ∑P(A∩Bn)  (概率的定义/出现可以 将空间/|Ω| 而得)  = ∑ P(A|Bn) * P(Bn)意义在于 如果条件概率好求 而 边缘概率不好算，可以这样做---分割方法很多：选择合理的。且离散函数在一个区间上的期望E的定义：E(X) = ∑xi * P(xi)  形式上相似，一对比替换那么：E(P(A|Bn)) = ∑P(A|Bn) * P(B) 期望的含义是小块的值---就是A被Bn分割的所有小块对所在Bn的平均占比、平均比率、占比期望值。所以A的概率还可以看作是一个期望值---A小块/Bn的期望值。

					>随机变量：随机动作产生的随机结果、随机情形、随机情况的某方面特征/显著特征/主要特征的度量。这种结果不能事先 知道所有条件 然后根据确定性关系精准计算出来、确定下来 是哪个结果哪个值；仅仅事先知道的是，所有可能的结果、值，以及这种随机 分摊到每个值/范围上的概率是什么、取到每个值/范围内/区域内的可能性有多大。对随机动作导致的随机结果的度量就是随机变量X

					>联合概率：事件A和事件B同时发生的概率P(A∩B)。没有带来信息，只是计算目标取值范围更小，已知信息里，随机变量定义好，分布概率确定，只是计算目标是：两个取值区域交集的概率 和 两个取值区域分别的概率 之间 的关系：因为两个事件时且的关系。
					>条件概率：事件A发生后事件B发生的概率P(B|A)。带来了信息已知信息不仅有A、B事件的背景，也有事件A已经发生 这个确定性 信息 导致的 随机变量取值范围的进一步 缩小，那么此时 随机变量在这整个取值范围内的概率分布  就要重新计算了---确保总概率是1，所以计算 出现在其中某一个指定区间/范围的概率---各处的概率累加起来。----所以新条件、新信息的加入导致的是 随机变量 的概率分布发生变化(无论是取值范围，还是各处上的取值概率)，比如样本空间范围收缩。。如果事件发生时仍然以原来的概率分布来计算，A、B事件(随机动作-随机结果)都是 在一个大取值范围内的两个小取值范围，那么 P(A|B) = 取值情况数(A∩B) /  取值情况数(B) = 数(A∩B) / 总数 / (数(B)/总数) = P(A∩B) / P(B) 交集是个变量，所以P(A∩B) 不能分解；这个关系连接了三个不同类型的概率。同理可得：P(B|A) = P(B∩A) / P(A) 而 P(B∩A) = P(A∩B) 所以 整合两个关系有：P(A|B) = P(B|A) * P(A) / P(B)

							>Landon的推导：使用定义：随机变量X的分布为h(x), 扰动随机变量E的分布为f(e) 那么新的随机变量Z = X+E 的分布为：g(z) = ∫f(e) * h(z-e)de 对h(x) 做泰勒级数，其中x = z -e并在x = z处取起点值，那么 h(x) = h(z) + зh/зx * (-e)+ 1/2 * з^2h/зx^2 * e^2 + ...  带入原来的式子 ,以及e的对称分布，e_ = 0 可以最终得出： g(z) = h(z) + 1/2 * з^2h/зx^2 * e_^2 + o(e_^2) 
									 然后，对于h(x)其实也是随机变量X的方差σ^2的函数，X ~ h(x, σ^2) 所以h(x,σ^2)可以按照σ^2来泰勒展开，假设新的随机变量x' 并没有改变分布模式，而根据x'的方差定义，从卷积的交换中先算dz积分而后算de积分可以得出D(z) =D(X) + D(e) + 2E(X)E(e) = σ^2 + e_^2 因为E(e) = 0这个随机误差是对称的, 假设的没有改变分布模式，也就是x'仍然满足h分布，只是方差变了，即g(x, σ^2) = h(x, σ^2) 即X' ~ h(x, σ^2 + e_^2) 对于这个新的分布函数，让它按照方差为变量 而在σ^2处泰勒展开，那么就可以得到 h(x, σ^2 + e_^2) = h(x, σ^2) + зh/зσ^2 * e_^2 + o(e_^2) 同样带入g(z)的定义表达式∫f(e)h(x,σ^+e_^2)de中，可以得出新的分布g(x, σ^2 + e_^2) = g(x, σ^2) + зh/зσ^2 * e_^2 + o(e_^2) = h(x, , σ^2) + зh/зσ^2 * e_^2 + o(e_^2)

							>随机变量相关：比如自己和自己：2X 它的分布就是以前取x0位置的概率，现在在2x0取到，所以是拉伸了。同理1/2 * X的概率密度是收缩了。2X = X + X
							>随机变量独立不相关：X1,X2...：比如X= (X1 + X2)/2  考虑简单情形，假设X1,X2的分布函数相同，为f，那么X1取x0,同时X2取x1 ，此时X取(x0 + x1)/2,此处由x0x1两点的概率密度为f(x1) * f(x0) 而因为对称性，有很多位置 相加除以2都是(x0 + x1)/2的位置，所以此处的概率是：∫f(x-λ)f(x+λ)dλ 其中λ ∈(-∞,+∞)，因为X1可以取任意位置，且任意位置确定后X2都可以确定且满足从而使得它们的平均为x位置---对x处的概率密度有贡献，并且从负无穷到正无穷才真正完全完备了x处的概率密度---使得等号两边值相等；此时真正抽象该计算公式，应该略去符号，而放在约束中，即：∫f(x+λ1)f(x+λ2)dλ 其中λ2 ∈(-∞,+∞),λ1 + λ2 = 0；才方便推广，此时有当n个独立同分布的随机变量时，概率密度就是：∫...∫∫ f(x+λ1)f(x+λ2)...f(x+λn)dλ1dλ2...dλn, 其中λ1,λ2,...λn∈(-∞,+∞) 且 ∑λi = 0。
								>独立推测: 1/n * ∑Xi 的概率密度：Xi是独立同分布的，所以等价于，一次实验中，取了随机变量X的n个实验结果，而显然这n个实验结果必然符合X的分布规律，即X的概率密度大的区域结果数量更多，而概率密度小的地方就寥寥无几甚至没有，所以当n的数量极大，看作无穷时，这个计算式1/n * ∑Xi  可以划分出若干个区域，每个区域的分布概率对应着点数，显然对于n近似无穷时，分布概率哪怕只是0.1的差别都是无穷个点的差别，如果划分的小区域的值近似相等，那么1/n * ∑Xi = 1/n ∑(m1x1_ + m2x2_ + ...) 其中mi代表点数，进一步转换为:1/n * ∑Xi = ∑(m1/n * x1_ + m2/n * x2_ + ...) 其中显然当n趋近无穷时，mi/n就代表了x1_处的概率，从而整体的含义就变成了X的期望E(X)的积分计算的定义的含义:∫x*f(x)dx，也即：1/n * ∑Xi = E(X) 所以一次实验所计算的平均值在n-->∞时就代表了X的期望。那么k次实验，每次计算的期望有所不同，但是因为计算的是同一个量，有理由相信，计算的k个期望是误差随机变量的k个具体值，即k个期望符合误差随机变量的分布规律:正态分布规律；所以说1/n*∑Xi 在n-->∞时服从的规律是正态分布规律。显然，期望 就是正态分布函数的对称轴x=μ中的μ。。
									进一步，由这一次实验的n个结果，还可以计算方差，显然就是X的方差，而k个平均值 也可以计算方差，自然这个方差就是这个误差随机变量的方差--是估计值(因为k个平均值结果还可以计算个平均值出来)；X分布函数方差和 正态分布方差的 实际联系的代数推导：类似，一次试验中n个实验结果，计算方差：1/n * ∑(xi - E(X))^2 同样，在n趋近于无穷时，因为点很多，会在区间内聚集，假设分为若干个区间，在这些小区间上 的值近似相等，那么计算公式变为：1/n * ∑mi*(xi_ - E(X))^2  等值转化为∑mi/n*(xi_ - E(X))^2 而mi/n显然就是X取值xi_的概率，而整体的含义就符合了X的方差的积分计算的定义的含义：∫f(x)(x-E(x))^2dx,当n趋近无穷时，它的值就是X的方差的积分计算的值：即有：1/n * ∑(xi - E(X))^2 = D(X) 而这个计算公式和误差随机变量的方差的估计公式只有均值 被替换为E(X) 的区别 和n没有趋近于无穷的区别，而计算形式是一样的，所以有理由相信X的方差D(X)才是误差随机变量的真实方差σ^2。
									>推论：n个独立同分布的随机变量Xi的均值随机变量 1/n*∑Xi服从的是N(E(X), D(X)) 这个正态分布。

								>从方差形式定义上看，是(X-E(X))(X-E(X))的期望---即和期望的距离的平方的期望，之所以要平方-因为根本上相看和期望的距离的期望，距离为|x-X_|但绝对值定义 在计算中不太方便，而不失先减后增的单调性 对绝对值加个平方 就消除了绝对值，形式变为：(x-X_)^2，同样可以描述和期望值的接近程度，而它的期望就可以描述这种接近程度的平均值了,另一种说法： 随机变量X离其期望的距离的概率分布的期望值(距离只为正)。从而在二维随机变量中，就自然引出了X和Y之间的协方差。
							>协方差：独立的两个随机变量X、Y，之间可能是相关的、无关的，比如线性相关的。
								>可以根据定义而 写出 协方差的概率密度的积分表达式：c(k) = ∫f1(x)*f2(±k/(x-E(X)) + E(Y))dx 其中x≠E(X)，且c(0) = f1(E(X)) + f2(E(Y)) 协方差概率密度计算推导中，看出k就是x到E(X）和y到E(Y)的距离的有向面积：k=(X-E(X))(Y-E(Y)),对应的概率密度就是取这个距离面积时候的概率密度，期望就是取得 距离面积的平均值。
								>本质：X，Y的协方差分布 ，自变量的每一个正数对应的概率 就是 X、Y 期望的同方向差值积的 等积位 的 概率乘积的和，一个概率和的大小：一般是高概率处的X和高概率处的Y对应的差值积代表的等积位上的概率和更大。。同理，协方差分布的每一个负值对应的概率是 X、Y对期望的方向构成 反方向 时的差值等积位的概率乘积的和，同样概率和的大小也一般是： 高概率X 处和高概率Y处同时相对期望方向相反 的 差值等积位 对应的概率和大。所以从协方差分布来看，值高的地方 也是高概率X和高概率Y对应各自期望差值积 的地方。X和Y越紧凑 ，协方差分布也越紧凑。协方差的倾向问题：假设协方差分布上，有-x 和x两点，那么对应的概率的计算过程，就包含了X和Y的分布上,相对E(X) 的±sqrt(x), 对应概率分别为a1, a2, 和相对E(Y)的±sqrt(y) 对应概率分别为b1, b2, 四个点的概率的计算，即+x上的概率包含：a1b1 + a2b2  , 而-x上的概率包含：a1b2 + a2b1 假如认为这两个概率值占据了-x和+x处的主要概率，那么它们之间概率的大小就表示-x和x处的概率大小，而x是一般的，所以表明协方差分布倾向于正还是倾向于负，；因此计算：a1b1 + a2b2 - a1b2 - a2b1 等于 (a1 - a2) (b1 - b2) 这个就是倾向性的判别式了，因此如果它>0 那么 协方差的期望为正，即(a1-a2)(b1 - b2)>0 即 高概率的X和Y的期望附近，X和Y的单调方向相同，表示正相关；同样，它< 0 协方差的期望为负，即高概率的X和Y的期望附近，X和Y的单调方向相反，表示负相关，为0则表示随着x的变化而 它的值 自由均匀的有高有低---或者因为概率值都为0，表示X和Y没有相关关系。 
								>推广：相关系数：cov(X,Y)/(sqrt(D(X)sqrt(D(Y)))) 如何证明相关系数一定在[-1,1]之间？ 并且可以表明X和Y的相似性？而类似这种计算，而发觉 两个向量的夹角公式类似---而两向量的余弦夹角值是否可以作为两个向量相似的判定？
								>从样本方差、协方差来看协方差的本质：样本和均值的差值表示为一个向量，那么这个向量：X = {Xi - X_} 同理Y = {Yi - Y_}
									>S^2 = 1/(n-1) *∑(xi - x_)^2  = 1/(n-1) * (X * X)
									>Sxy = 1/(n-1) * ∑(xi - x_)(yi - y_) = 1/(n-1)*(X*Y)  协作产生的方差，看作是协方差。
									而此形式已经是向量的点积了，而自然想到向量的夹角，它们的夹角会体现什么性质--两个向量的方向相同----至少说明并向变化， 即有一种相关性，或者说和两个向量的模长无关，那么可行的处理就是除以两个向量的模长，即Sxy/(|X|*|Y|)， 那么显然，从向量角度看，这个值就是两个向量的余弦值了----因此值域在[-1, 1]之间，成为相关系数 而|X| = sqrt(D(X)) 所以Sxy/(|X|*|Y|) = Sxy/(sqrt(D(X))*sqrt(D(Y))) = Sxy/(SxSy) 也即相关系数就是余弦距离----线性相关程度。
										>X,Y两个向量构成的n个二维平面点，体现的是差值的分布，即随着横轴方向的差值的变化，而纵轴方向的差值发生了怎样的变化，有什么统计规律---即大部分情况是怎样相关的---如正相关或者负相关。比如只有两个点，如果先高后低，那么说明x轴方向差值的增大，而y轴方向的差值减小了，说明是负相关；同理如果是先低后高，那么说明随着x轴方向的差值的增大而y轴方向的差值也增大，这就是正相关；(但是涉及负数就难以解释，所以暂时不引进负值)；加进1个点，2个点，更多的点，n个点，那么形成一个点的分布图，此时更可以看出横轴和纵轴差值的相关程度：分布越是在一条线的附近----那么相关系数的绝对值越大，相反，分布越是随机、范围宽，相关系数的绝对值就越小---乃至于完全随机分布、点集关于期望中心对称分布的0;;所以 相关系数其实描述了点的线性集中/分散程度----越线性越集中绝对值越大。

						>是最佳估计的判断标准：误差的平方和最小。原因：表面为实际经验，真正为误差总是 随机误差 ---随机误差正态分布，均值为0，随机误差的方差描述了观测值和真实值的接近程度，方差越小则越接近，而方法的关键就是计算误差平方和---且正相关，所以越接近真实值则要求误差的方差越小，而方差越小则要求误差平方和越小；从而误差平方和越小越接近真实，反过来，如果把一次观测当作是可信的最接近真实的，那么最接近的真实 被求出来的方法就是 满足 误差平方和最小的那些函数参数。

	>一口价博弈问题：
		>已知(情况，事实)：商家和买家交易一样物品。商家和买家对商品都有一个估价vs, vb ，都有一个出价ps,pb 其中出价是估价的函数，ps=f(vs) , pb = g(vb);假定vb,vs都在[0, 1]上均匀分布; 双方互不知道彼此的报价，成交条件是pb>=ps, 成交价格是：(ps + pb) / 2 ; 而前述知道ps,pb,vs,vb都是随机变量
		>目标要知(目标关系)：商家的期望利润最大时的出价函数ps, 以及买家期望利润最大时的出价函数pb ， 以及两个出价函数是否可能互为最佳出价函数--纳什均衡？
		>利润可以表达为的式子： 成交概率 * 成交条件下商家获取的利润(ps待知) + 不成交的概率 * 0(没有利润)  所以数学表达为：对于商家的利润：下面会求。
		>分析情况中各对象满足的、符合的、适用的、所属的根据(定义，一般概括性判断，约束、限制，原理、定理):
		>可知：对vs,vb求P(vb>=vs)假定商家出价x 那么买家出价必须高于x才能成交，高于x的概率是1-x(因为均匀分布), 而商家出价概率就是dx, 所以P = ∫(1-x)dx = 1/2 ;;;第二，对于模型y = f(x) 如果x在[0,1]上均匀分布，f是单调递增函数，那么y的分布计算方法：假设当y=k时，则x = f(-1)(k) ,则dk宽度的y对应到x上的宽度为: f(-1)(k+dk) -  f(-1)(k) 因为均匀分布所以宽度就是概率，因此dk上的平均概率是：(f(-1)(k+dk) -  f(-1)(k))/dk = (f(-1)(x))' ;;;第三，假定ps,pb都是单调递增函数，如果商家出价k, 那么商家期望利润 = ∫P(pb>=k) * ((E(pb) + k)/2 - vs) * (f(-1)(k))'dk
		>枚举计算：假定买家的战略/函数pb是：当vb>=x时pb=x ，当vb<x时pb=0； 那么对此，商家的战略：在自己vs估值[0,x]区间上，如果出价<x那么一旦对方估价>x而出价=x，则成交时没得到多少利润，而如果出价x那么也成交且得到最大利润x，而出价>x则不会成交而损失机会，所以商家选择出价x; 当商家估价vs>x时，出价因为递增，所以必须>x，而此时都不会成交，所以会出1。同理，在此条件下，如果再看买家的策略，那么也是[0, x]上为0，[x,1]为x为最佳策略。。。如果定义vb>=vs是有利的交易(不管成交还是没成交)，那么其中成交时的交易概率占有利交易的比例：称为交易效率： η = P(pb>=ps|vb>=vs) / P(vb>=vs)  用估价交易图(平面直角坐标系下看)可以展示出来。
			  假定商家的战略/函数ps是：ps(vs) = as + cs*vs ，那么ps服从[as,cs]上的均匀分布，；那么对此，买家的战略：首先交易条件显然要求买家出价pb在[as, cs]上才行，而自己的利润将是： ∫ (vb - (pb + E(ps| ps<=pb) ) / 2 ) dpb = ∫ (vb - (pb +(pb + as) / 2)/2)dpb 用概率来等效表达则为：商家整体的期望出价成交利润为(不含成交条件，一般情况下) = (vb - (pb + E(ps|ps<=pb))/2) * P(pb>=ps) =  (vb - (pb +(pb + as) / 2)/2) * (pb - as)/cs 用求导方式求出导数为0的条件为：pb = 2/3vb + 1/3as 同理的方式可以求出商家的最优反应ps的精确表达式：。。。。再用ps<=pb的交易条件得出 vb,vs的关系而作为交易成交的条件，从而得出交易效率---利用估价交易图。
	>利益的合理分配问题：n个人内部互相2人及其以上形成若干个组合s，每个组合因为内部合作产生了利益v，问当n人合作时，产生的利益vn如何分配：---每个人各占多少是公平的。
		>某个人i的Shapley值的计算：分为几步：先得含有i的所有组合si；对于所有组合中每个组合sij，计算该组合产生的利益v(sij), 再计算该组合去除i元素后的组合的利益值v(sij/i) 自然可以得到因为减少元素而带来的利益减少值v(sij) - v(sij/i); 计算sij的的元素个数|sij|，计算元素个数函数w(|sij|) = (n - |sij|)! * (|sij| - 1)! / n! 其中n就是n个人的总量n，此值像i元素占所在集合的权重值;  然后自然而然计算 因i而组合损失元素统一指标值φij: =  因i而利益减少量 * i占集合权重值 = (v(sij) - v(sij/i)) * w(|sij|) 最后计算总共损失统一指标值 φi = ∑φij ； 而φi值就是Shapley值，就是每个元素i所占的比例----利益分配值。。
		>使用方法：n个人合作产生了效益，首先必须明确效益是什么？怎么度量：比如说创业产生的价值，那么这个价值量可以当作效益，同时价值量比各自单干多的量也可以当作效益值；又比如合作减小了亏损\耗费，其实也是合作效益，那么这个效益的度量就可以是比单干减少的亏损量\耗费量----最后真正耗费量就是单干-这个因为合作带来的减少量， 当然也可以直接是合作时的亏损值\耗费值---越小则表示负担\分配就越小。
		>Shapley值的缺点和新办法：缺点是需要的数据v太多--2^n-1个，而实际很难满足；而如果只根据n人总效益B， 缺第i人时的总效益bi 这n+1个数据而得到分配因子：则是新的方法5种：
			>协商解：假定已经计算出了各自分配方案xi 那么∑xi 和B之间可能有差值，这个差值应该均摊，Δxi = 1/ n * (B - ∑xi )  而计算值∑xi定义为： ∑xi - xj = bj 那么根据这n个方程而xi为未知量从而 直接相加而 得出∑xi的表达式：∑xi = 1/(n-1) * ∑bj 从而 xj = 1/(n-1) * ∑bi -bj  再加上分摊的误差Δxi 那么xi_ = xi +Δxi  = 1/ n * (B - (1/(n-1) * ∑bj) ) + 1/(n-1) * ∑bj -bi
			>均衡解：假定各方接受分配最低现状值已定：d = (d1, d2, d3....dn) 目标分配x = (x1, x2, x3....xn)  且∑(xi - di) = 恒定正值--都一点 ；那么分配差值的积∏(xi - di) 越大表示越均衡---比如都相等的时候。因此计算max∏(xi - di) 即可
			>最小距离解：假设理想分配上线已定：x_ = (x1_, x2_, ... xn_) 目标分配x = (x1, x2, x3....xn)  那么目标分配应该使得各自到理想分配的距离(xi - xi_)^2之和最小：min∑(xi - xi_)^2
