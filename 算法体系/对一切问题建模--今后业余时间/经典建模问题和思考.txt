					>泰勒级数的自然推导过程：在导数定义之后，x + Δx处的函数值f(x + Δx) 可以近似表达为：f'(x) * Δx + f(x) ,显然，当Δx增大之后，这个误差是增大的。而增大的原因在于x到x +Δx过程中，f'(x)的值发生了变化，所以，为了减小误差，精准的Δy 计算应该是：将Δx再分割为n份，每一份乘以其所在处的导数值，然后求和：f'(x)Δxn + f'(x + Δxn)Δxn + f'(x + 2Δxn)Δxn + ..... 而显然，从第二项开始，后一项可以用前一项表示出来，因为 f'(x + Δxn) - f'(x) = f''(x)Δxn, 从而第一轮替换，就可以得出用一阶和二阶导数表示的精确表达式，而且因为nΔxn = Δx, 再估计f''(x + kΔxn) = f''(x) ，和n(n - 1) Δxn2 = Δx2 那么可以得到一阶和二阶来表示的较精确表达式：Δy = f'(x) Δxn + 1/2 * f''(x)Δxn  。但是这个也是不很精确的，因为对f''(x + kΔxn) 做了简化估计，而实际上应该再次精确表示，即用导数定义而f''(x + Δxn) - f''(x) = f'''(x)Δxn 那么同理可以得出用三阶表示的精细表达式，而在同样估计式的计算过程中，发现f'''(x)的系数是n(n -1) + (n - 1)(n - 2) + ...这样的连续求和，完全是一个数论问题，而通过转化为自然数求和问题--和1 + 3 + 5 = 3^2这样的自然数平方和和自然数求和的关系，和转化为面积和问题，可以得出平方和公式: (n - 1)(n - 2) + (n - 2)(n - 3) + ... = 1/3 *(n - 2)(n - 1)(n),
							         从而得出用1-3阶导数表示的较精确表达式：Δy = f'(x)Δxn +1/2 * f''(x)Δxn^2 +1/(2 * 3) * f'''(x)Δxn^3  看到这个形式之后，以及想到估计时做的动作，如果用4阶乃至n阶导数来表示时，形式应该是很自然的推广：即Δy =  f'(x)Δxn +1/2 * f''(x)Δxn^2 +1/(2 * 3) * f'''(x)Δxn^3  + 1/(2 * 3 * 4) * f''''(x)Δxn^4 + ... + 1/n! * f'''(x)Δxn^n    而且估计到4阶进行的计算中，还自然推论出数论公式： (n - 1)(n - 2) * ..*(n - k) + (n - 2) * ..*(n - k - 1) + k * (k - 1) * 2 * 1 = 1/ (k + 1) n * (n - 1)(n - 2)(n - 3)...(n - k + 1)  。。。而显然的是，给定一个Δx，无论多大，只要 n足够大时--到足够高阶时，都能保证 Δy的值是正确的。所以，如果x的值取0，而Δx取任意x，那么就得出了y = Δy = f'(x)Δxn +1/2 * f''(x)Δxn^2 +1/(2 * 3) * f'''(x)Δxn^3  看到这个形式之后，以及想到估计时做的动作，如果用4阶乃至n阶导数来表示时，形式应该是很自然的推广：即y = f(0) + f'(x)x +1/2 * f''(x)x^2 +1/(2 * 3) * f'''(x)x^3  + 1/(2 * 3 * 4) * f''''(x)x^4 + ... + 1/n! * f'''(x)x^n   n --> +∞ 这就是泰勒级数。

								Xn的求解，可以从e^(jｎωt)函数本身性质中找，消除更高频的项是关键，现在知道的性质是, 一个周期T内：∫ e^(jｎωt) = 0 始终成立，而为了将Xk和指数函数分开，显然可以对f(t)乘以一个 e^(jkωt), 而其他项乘以 e^(-jkωt)后仍然保持指数形式，再利用之前的该指数函数积分为0性质，对剩余项在T内积分，自然就消除了，而Xk项不会消除。即∫ e^(-jkωt) * f(t) * dt = ∫ Xk dt = T * Xk 那么有 Xk = 1/T * ∫ e^(-jkωt) * f(t) * dt  ; k : -∞ --> +∞, k≠0 那么就只剩下a0的求解了，很明显一个周期T内： ∫ f(t) dt = ∫ a0 dt = T * a0 从而 a0 = 1/T * ∫ f(t) dt  从而有: f(t) = 1/T * ∫ f(t) dt + 1/T * ∑∫ e^(-jkωt) * f(t) * dt * e^(jnωt)  ；ｎ：-∞--＞+∞ n≠0 这就是著名的傅立叶级数的指数形式。
							>傅立叶变换的自然引出：上述求解Xn中，n取的每个值都要计算，比较复杂，而三角级数形式又和积分表达式的差分形式类似，而且三角级数形式表示中，其实可以看作是一个积分求面积的过程，把n当作自变量，那么相当于Δn = 1, 而显然当如果假设频率是连续的---即允许是小数--只要单位上放大其实可以近似是小数，那么这个傅立叶级数的连续形式就是：f(t) = ∫ X(ξ) * e^(jξt) dξ  这样，ξ把频率也包括进去了。那么此时自然关系的是X(ξ) 这个振幅随ξ的变化规律，即当频率在ξ时的振幅X(ξ)是多少。。而类似上述积分方式：这里也可以积分方式：X(η) = ∫ e^(-jηt) * f(t) dt 这个就是傅立叶变换。其中系数和一般的有区别，但是整体不影响。

			>光波振动方向：光子来看，和运动方向垂直。但是对于自然光束来说，是很多光的混合，不仅波长不同，而且同一种波长的光，振动方向各个方向都有--即和传播方向垂直的方向都有，如果要分解出某个振动方向的波，可以让光束通过一个狭缝--只允许单光子通过，显然当振动方向狭缝方向相同时可以穿过狭缝，不同则因会被阻挡。一束光的反射光和折射光垂直时-反射光是线偏振光，而折射光是部分偏振光（即在振动方向外的其他方向还是有尽管弱的振动光）。散射光也是偏振光。偏振光的特性在于无论光沿着什么方向传播，其偏振方向不会变---比如竖直和引力方向同则一直都和引力方向垂直，同理水平----从而3D电影偏振片和眼镜偏振片---无数平行紧邻小狭缝。
			>光子的正确理解：是一个在运动方向垂直平面上的某个固定直线方向一直来回振动的粒子，且振动的振幅体现能量，而波长和频率固定--且确定传播速度，且不是严格的在一个方向直线传播的，而是在传播方向上有轻微的不确定--轻微的扰动--小概率性的偏离---且显然和传播距离有关--越远就越不确定了---偏离原因可能和时空性质有关：时空对里面运动的物质实体的影响和约束限制干扰。
			>光折射和反射的根本原因：是因为在介质表面存在着张力，合张力方向垂直介质表面，张力让光在介质表面内侧即将出射时受到内部吸引力阻力，方向垂直介质表面向内侧，从而发生反射，又光速太快而阻力冲量太大而发生弹性碰撞，所以反射角等于入射角，而出射成功入射到另一种介质表面上不内不外刚好会受到最强的张量作用力--即一半在内一半在外，这是看哪种介质的合张量冲量更大，偏折方向自然就会折向哪边，从而发生折射；且介质对光的吸引力大的更会阻碍光的传播而滞留光，使得光速减小。。

		>数学理论：
			>变分问题：一般模式：求使得一个定积分取得最值的积分函数应该是什么样子。。其中，这个定积分中，被积函数是一个f(x) 和 它的各阶导数等和它相关的衍生的量的表达式。而这个由泛指函数构成的被积表达式构成的定积分 称作 泛函，，被积函数叫做 核。（泛指函数构成的函数）（区别于定函数在于：定函数的表达式的构成元素都是确定形态的基本函数和常量和基本运算构成的）(指数函数、三角函数可以当作时基本运算)
				>特点1：输入函数定，定积分的值确定；输入函数可以任意函数。
				>特点2：函数的微分是自变量临近微小变化时函数值的变化量；那么在泛函中，被积表达式中的独立函数的临近微小变化而导致定积分值的微小变化量叫做泛函的变分。
					>问题：泛指函数的临近变化是怎样的另一个泛指函数？泛指函数是一种运算(基本运算、基本运算组合构成的运算，新的非基本运算的新定义的运算---一种新映射)，泛指函数可以形象化为某区间上的任一条光滑曲线，泛指函数的集合就是某区间上所有光滑曲线的集合。泛指函数的集合称为赋范线性空间， 值的集合叫做数空间。
						>泛函的一般形式： J = ∫F(x, y, y' ) dx   而其中y = y(x)  。但是同时，F本身却可以看作是F(x, y, z)即三维直角坐标系下的一个函数，只是,z和y都是x的函数都相关而已。从而变分：F(x, y + δy, y' + δy') 看作是F(x, y, z)附近的一个值，而这个差值正好可以用F对y,z的偏微分表示，即δF = зF / зy * δy + зF / зy' * δy', 所以变分δJ = ∫[зF / зy * δy + зF / зy' * δy']dx  。。导数= 0是函数极值的必要条件，而导数=0的本质是dy =0 即随着x的变化，dx处的dy无限接近于0，那么就取得极值了，同样的，变分=0也是泛函取得极值的条件。因为δy也是x的函数--且x = a, x = b处值为0（因为δy是一个差函数，而任意两条曲线都起于a终于b，所以差值在a,b处都为0），所以用分布积分整理得：δJ = ∫[зF / зy * δy + зF / зy' * δy']dx = ∫[зF / зy  - d (зF / зy') / dx ]δydx   很显然，δJ存在始终为0的情形--不管δy如何变化--即不管其他任何函数和某个函数有多大差别，只要这个函数满足这种性质зF / зy  - d (зF / зy') / dx = 0， 那么泛函的变分始终为0，即δJ = 0 恒成立。那么这个函数就是使得泛函取得极值的函数。这个等式就是E-L方程。如果把E-L表达式当作是对某个泛指函数对x求导的化简结果，那么这个泛指函数可以是：F(y, y') - y' * зF(y, y') / зy'    所以F(y, y') - y' * зF(y, y') / зy'  = C也是泛函取得极值的必要条件的一种约束条件形式。

					>原理：可以认为|假定y, x都是第三方变量θ的函数，而且y(θ) 和x(θ) 之间依然保持 y和x原来的定量关系，这是可以的，那么此时其中一个函数如y(θ) 或者x(θ) 就可以随便指定了---当然是为了让将微分方程化简的方向来指定---且必然还要保证值域保持不变--不缩小可以扩大；那么此时的代换：比如cotθ 或者tanθ。。。而具体指定的时候，却应该对微分方程的表达式结构中最明显的通过替换可以化简的部分进行指定-----不管那一项是高阶还是低阶。。这种方法来解悬链线的E-L方程，或者最速降线的E-L方程都是非常容易的。

					   >拉格朗日差值公式：来源：先是有n个点，设计一条曲线过着n个点且是一个光滑曲线，(x0, y0),......(xn, yn)   如果用一个多项式来表示过某个点而在其他点处为0----即不贡献值，那么每个多项式都在各自对应的点处贡献值，那么这个多项式的线性组合，就过所有的点，而在这n个点之外，每个多项式都会贡献值---来导致连贯，那么这是合理的。所以：L = ∑ λk * ∏(x - x0)...(x - x<k -1>)(x - x<k+1>)...(x - x<n>)   叫作这n个点确定的拉格朗日插值函数：拟合函数。而且显然，λk是可以确定的：因为对应的那个多项式在(xk, yk)不为0---有贡献--而且在其他多项式上没有贡献---值为0，所以有λk = yk / ∏(xk - x0)...(xk - x<k - 1>)(xk - x<k+1>)....(xk - xn)     带入前式简记得：L = ∑yi * li(x)  其中li(x) 就是 插值基函数。进一步的重心拉格朗日计算公式，仅仅是将  插值公式运用于对g(x) = 1 而得出的插值表达式 记录下来，用 拉格朗日插值公式除以这个1而已。
