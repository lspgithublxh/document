  >先知：
	##1 范数：@1 向量的范数：只和向量的各个元素相关。（绝对值之和范数，平方和的平方根范数，p次方和的p次方根范数，绝对值最大值范数，绝对值最小值范数）。。尤其2-范数（欧几里得范数，平方和平方根范数，性质：w<T> * w = (||w||)<2>,,范数的平方和和向量的关系。）
	    函数间隔：向量x处的函数间隔：就是f(x)的绝对值（其中f(x)就是自变量的线性组合,f(x) = w<T> * x + b），而如果加上可以获知的（对函数值的）结果判定（根据取值正负）而取值1or -1的y,则函数间隔可以表示为y* f(x)
			f(x)对于一个样本集合的函数间隔就是函数间隔最小的样本。但是函数间隔是没有说明意义的，因为k * f(x)，k可以很大，而对应的超平面并没有发生任何改变，但是函数间隔却大大的改变了，所以需要消除这个k的影响，而定义几何间隔如下：
	    几何间隔：任意点x（向量x）的几何间隔是点到f(x)的垂直距离，可以推导为：y * f(x) / ||w||（w不仅是线性组合的各个系数组成的向量，也是线性组合对应的超平面的法向量）,,这个正好表明了函数间隔和几何间隔的关系（相差一个范数因子）。
			把离超几何平面最近的点到超几何平面的点的函数间隔设定为1（即要满足这种形式的该超几何平面的表达式），那么该点到平面的几何间隔也确定了，且是最小的一个。同时发现一个特征，就是当直线在平行运动的时候，各个样本点到直线的几何间隔有时候是有“重复”的。因为有这样一个性质：两个样本点，当直线位于它们之间时，它们到直线的距离有最大值，就是直线垂直于它们之间的连线时（最小当然是经过它俩）；且认为位于它们之间且距离最大时（同时应当有它们几何间隔之积应当最大）是最好的分割。对于一对点的分割，这种定义是明显的，推广到很多个点的分割（两对，三对，n对），也要要求满足最简单情形具有的性质：
			1)直线在样本点之间，不能在所有样本点之外（即所有的样本值带进直线表达式里的结果都是<0或者都是>0）（否则几何间隔没有意义）
			2）几何间隔之和最大的基础上，几何间隔之积最大。
	    比较线性拟合：线性拟合是将样本点聚合，规律是误差平方和越小那么直线将更接近于样本点----误差平方和的集合解释就是函数间隔的平方和（同样的系数--，即样本点都用的是同一个系数\同一条直线，不存在放缩问题，有缩放也是每个样本都加，因为样本都属于同一条同一个形式的直线：都属于一条直线一个形式,所以带进去求最小值就有意义）。而SVM的，是将样本点分开，误差平方和的大小没有说明意义--越小就是将点拟合了，而不是分开;而越大则越远，也没有分开。	
	    点与超几何平面的关系：对于分类超几何平面，点到平面之间是有距离的，令两侧点到超几何平面的最小的函数间隔为1，那么几何间隔确定了，而且其他点到超几何平面的函数距离都将在[-1,1]区间之外，且该点带入函数所得值是该点的函数间隔正比例相关的关系。
	    点带入函数所得值和点的判定值：不同。点的判定值只有-1 或者1之分。
	    对偶问题：
	    KKT条件：其实就是下面所说的条件：全部的等式约束条件和不等式约束条件(y<i> * ( w<T>x<i> + b >= 1)（即“曲线”，“区域”） + 函数(f = w<2>）整合条件的各个偏导数为0.
			而各个偏导数为0，和等式条件就可以推出w<i> = -∑λ<i> * y<i> * x<ii>   。。其中负号可以省略，因为λ的任意性。其中，这是条件 = 1 时的结果，而如果大于1，即很多其他点，不影响计算（）-----因为从原函数表达式来看，此时这些函数间隔大于1的点λ的值必须要等于最小值即0时才能保证函数取最大值。。
	    支持向量点：即函数间隔为1的那些点。并且上述的w<i>的表达式中，经过上述的推理，其中的点都是支持向量点。其他点不会考虑进来。只是这些支持向量点确定----必须还要确定λ的值,所以关键就是求λ的值（而yi,xi都是确定了的）(λ定则w定)----SMO算法求解。		
	    拉格朗日函数：
			>预知1：集合和任意一个元素，是对立的不同的（正如一条直线和一个直线系）。比如一个函数取值等于0和取值等于一个区间[a,b]，各自所满足的点构成的集合（图形）是远远不同的。
			>预知2：求极值问题：已知一个函数f(x,y,z),求在φ(x,y,z) = 0时的最大值max。（本质上就是求函数在局部空间的最大值，而这个局部空间就是自变量之间满足的一个约束条件。而约束条件就是自变量之间的相等关系或者不等关系，一个或者多个）
				求解的方法之一是结合这些不等关系和相等关系，带入到f中，形成一种消元的情形，但是这不是可以一般描述的方法，并不一般。最好不用互相带入，而直接用原生的函数和约束关系而构造出新的函数，求解这个新函数的极值---就是局部的极值（把局部约束化入函数中）（一个自变量是点上的极值，两个自变量是线上各点的极值，三个自变量是平面上各点的极值，以此类推）（点用向量表示，约束关系是向量限定：所有类向量都满足的某种度量g 当这个度量值等于0时候的那些向量，而函数所有类向量都满足的某种度量f）。问题简要描述为：定区间，求极值。
				>一般的求极值的方法：导数为0的点和端值，从这些点中找仍然是最好的方法（对于连续可导函数）（计算机方法可以使用到）。
				>一般的带约束条件的求极值方法：将约束条件化到函数中（消元），从而转化为求一个更低维函数的极值。
				>只有一个全局最值点函数的最值规律：各个维度上的导数都是0（从函数整体上看,就是各个偏导数 = 0）
				>拉格朗日函数：本质上是统一表达了约束条件和函数---根据时上述求极值的一般方法的思想：偏导数=0或者端点中找极值，根据的第二个思想是：把约束条件函数φ(x,y,z)当作是一个整体变量h（相对于x,y,z都独立）,而且它的值恒等于0,所以这个整体变量没有导数，并且可以加一个系数λ（是变量，且没有影响）。而拉格朗日函数形式的统一条件和函数，当在求导数时，得到的偏导数方程还是原函数f的变量个，而加上那些恒等于0的那些“整体变量”（约束条件）方程，所以方程个数没有增加。只是表达上简单了。（整体求偏导数和分解求偏导数，函数分函数求偏导数和整体函数求偏导数）
			>预知3：函数的等势线和其他一条可导曲线的关系：
				如果一条曲线的某一处开始正在接近极值中心，那么该处的法向量不可能指向中心点更不会指向相反的方向（整个范围，必然指向连线的垂线的和极值点的同一侧），且必有该处和极值点的连线，与该处的前进方向（切线）的夹角小于90度。而如果正在远离，那么这个夹角将大于90度，而如果正在接近和远离的转折上，这个角度是90度，而条件已经明确曲线连续，所有前进方向的转变是连续的（正如导数的连续一样）,所以必然有90度（只要该角度有大于或者小于90度的情形）。而这个角度能否一直大于90度或者小于90度或者保持某个大于90度的角度？由于前进方向始终和与极值点的连线在同一条直线上，那么这个角度不能保持，只能一直不断变化，能否形成一个大于90度的周期？远离极值点是可以的，但是接近极值点就是不可以的。如果在曲线在正负无穷区间，那么必然有接近的时候，而该角度是连续变化的，从180度再到180度，中间必然经过了90度。如果曲线不在正负无穷之间，那么它在改变方向的过程中，角度连续变化了180度，那么中间必然有一时刻角度刚好是90度。
				所以：导数连续的曲线必然会和等势线相切。
				>推论1：导数连续的曲线，对于一个有极值点的导数连续的函数，那么曲线和函数的等势线至少有一个切点，但这些切点中不一定有和极值中心最近的点（参见二次曲线和内侧的极值点的椭圆等势线的特殊关系），但是有极值点，因为曲线和等势线同样有接近和远离（也可以用切线、法向量来分析）。
				>推论2：离极值中心最近的点，不一定是曲线的极值点。因为等势线的原因。如果曲线就是等势线，且等势线为椭圆，那么最近的和最远的都是一个函数值。但曲线不是等势线时，那么和等势线相切的点中一定有极值，只是最大或者最小未定而已。
			>预知4：函数的等势线的法向量的求法和曲线的法向量的求法：
				首先， 切向量必然是对向量的求导数的结果，而对向量求导，就是对各个分量求导，就是求偏导数。而法向量和切向量的乘积为0。
				所以：曲线相对于函数的极值的求法：令函数的等势线的法向量 等于 曲线的法向量乘以一个非0常量λ，那么求出的点中就包含了最值点。
				      这个式子处理一下就是：对f + λg 的全部偏导数都为0，，即各个偏导数为0。有一个专门的符号：倒三角符号。
				推论1：求函数的极值是求函数的各个偏导数为0.而上述表明了，如果想求一条曲线g关于函数f的极值是求 f + λg 的各个偏导数都为0.而g就是一个约束条件。推广：如果含有更多的条件，那么就是求f + ∑λ<i>g<i> 的各个偏导数为0，如果取不到就取端点值（前提，条件必须相交，没有交点则没有解）。（高维度）（平面也是）
			

	    拉格朗日方程：
	    最小作用量原理：（堪称和熵相比的普遍意义）
	    核函数：高斯核函数-多项式核函数--线性核函数。
		    将样本点从输入空间映射到更高维度的空间，而此时样本点之间是可以超平面分开的，定此超平面则成（或者可以将此超平面化为非线性低维空间函数）。
		    >映射的可行性：点确定，且判定值确定..但是新输入的样本，只知道点不知道判定值。
		    >映射的方法：1）映射矩阵（变换矩阵），比如3 * 2的矩阵可以将一个2 * 1的矩阵变换为3 * 1的矩阵。
				 2）预知1：坐标系和坐标值：
					 坐标系的本质：向量（向量可以描述一个坐标系<如直角坐标系>）
					 坐标值：具体的某个向量
				    预知2：构造坐标系：即构造一个向量（很简单，指定维数就可以了）
				    预知3：表达式的形式和坐标系（向量）的关系：
					   @1 不仅表达式中的各个变量可以看做向量的分量（坐标系的分轴的值）
					   @2 表达式中的各个变量的运算组合整体也可以看作是向量的分量
					 ---举例：一个函数的通式，往往可以分解出多个整体，而每个整体可以当作一个向量的分量，向量的维数就是整体的维数。
						  这样，比如一个两个变量的二阶函数展开，那么系数可以看做是二项式的系数，而每个相加因子的变量部分可以当作一个向量的分量，所以三个运算整体可以看做是一个三维变量的三个分量。。而用这个3维向量来表示二项式时，就会是一个线性组合的形式。
					    结论：非线性表达式转换为线性表达式的方式：将各个运算整体都当作是一个变量，运算整体之间保证是线性组合关系，而这些变量统一是一个新的高维度向量的分量，所以非线性表达式变成了一个高维度向量分量的线性组合。
                     		 3）核函数定义：两个向量的核函数等于两个向量的内积 + 1 的平方（这个平方结果可以看作是系数乘以高维向量的结果）。。。核函数是一种计算方法，一种比较简单的将向量从低维空间映射到高维空间的方法（一个向量自己就可以,一维到2维，2维到5维（3维其实足够），3维到19维）。
						 内积是点乘（而不是叉乘）。
						计算核函数的目的：计算λ（尤其非线性的分开器）。
	    变量个数同样本维数的区分函数：
				       一维样本为例子：线性函数ax + b只能将x轴分为一个点分开的两部分；
						      但是非线性函数 (x - m)<2>能将x轴分为线段和线段之外的两部分。该二次函数按级数展开：ax<2> + bx + c 。可以看作是(x2, x)二维变量的分量的线性组合。所谓区分函数，即样本带入函数之后，映射到了不同类的值----这些值刚好可以分为2类，比如大于小于0，比如大于小于-1.
								且可以把x轴上的点，对应于x-y坐标系上非线性函数的点（线性函数也可以），看作是x轴上的一维点映射在x-y二维平面（空间）上的点（同理，只要愿意，还可以映射到三维空间中的点--只要分离函数有）。
								（问题就是如何求这些系数）----已知样本和样本的判定值（即每个样本属于2类中的哪一类）..但是完全不知道两类样本之间的轨迹是什么形状。
						      同理，对于二维样本，ax1 + bx2 + c 只能线性分割为两部分。而假如有一个非线性函数f的按级数展开式是：ax1 + bx1<2> + cx1x2 + dx2<2> + ex2 + f ，可以看做是（x1, x1<2>,x1x2,x2<2>,x2）五维向量的分量的线性组合。
