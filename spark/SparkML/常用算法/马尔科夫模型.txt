   	>马尔科夫过程：
		     假设有一个集合E,其中元素都是整数，如果有一个随机序列，序列中每个元素从E中取，而且随机序列在构造的过程中有：第i个元素出现为什么元素与前面出现过的全部i - 1个元素是什么都没有关系。
		     推论1：随机序列中的元素可以重复。
		     推论2：随机序列中第m + n个元素出现为j的概率，等于前面n个元素为任意元素时第m + n个元素出现j的概率。即第n个元素出现什么，会影响到底n + m个元素出现什么，但是不会影响到之间的m - 1个元素会出现什么。。这是一种间隔影响、跳跃影响。
		     推论3：如果上述中与m无关，即m为其中1步到n步中的任何一步，这是齐次马尔科夫链，那么从元素i到元素j的概率p<ij>有全部种可能，与中间到达某个元素的概率，和再从这个元素到j的概率都有关系，且（某个元素也是任意的）。且i是什么，j又是什么，是什么都可以。
			    而这样p<ij>就可以形成一个矩阵，叫做转移矩阵。对于元素0-M的正整数序列为集合的例子，如果每次转移只能是相近的元素，即转移到相邻的整数，那么这是一个一步转移矩阵。

		     >隐马尔科夫链：
		     		>转换概率：开始到了一个状态，然后从这个状态直接的不经过其他概率性操作动作而概率性的到达下一个状态，这个概率就是转移概率。显然，下一个状态有多少个，这个转移概率就有多少个。且这个状态是隐含状态。
				>输出概率：到达一个隐含状态后，再让这个状态所代表的具体事物产生一个具体的输出，而输出是多种的，每种有一个概率，成输出的每种是一个输出状态、可见状态，每种的概率是输出概率。
			 	 推论1：构造一个隐马尔科夫链：
					已知一个起始隐状态，已知各个隐状态之间的转换概率，又已知每个隐状态输出各个状态的概率（即输出概率）,那么自然可以产生一个隐马尔科夫链----即这些信息足以产生一个马尔科夫链。（充分）
				 推论2：已知序列，各个隐式概率和输出概率，求最可能的隐马尔科夫链。
					1) 那么从最后到最初：开始：转换概率 * 输出概率 中最大的那组作为最后两个隐马尔可夫链的节点。再不断向前推。
				        2) 因为开始状态确定后，后面就可以确定了，所以可以分别计算开始状态为全部可能的隐状态，然后依次往下计算下一个节点出现某某的概率，这样推到结尾，得到隐状态数目个概率链，求乘积中最大的那个就是最可能的了。
				 常见的问题：输出的序列是容易观察到的，但是隐状态是难以观察到的，所以想通过易知推测难知。
					     此时还需要：转换概率和输出概率。
				 -------------这就是常见的问题模型。且：输入概率阵、转换概率阵、输出概率阵，都是明确的。

				 推论3：已知序列k个，每个序列长l，求最开始的状态概率向量(N,1 * n)和隐状态转移概率矩阵(T,n * n)和可见状态输出矩阵(U,n * m)。
					有k个，可以推知这个序列各个位置各个可见状态的概率，共：m * l个概率。利用这个可以推知（理论上）隐马尔科夫链的上述三大参数矩阵。（条件之一：l >= ( n + n * n + n * m) / 3）。
					---因为有以下规律：
					1) 第k次时隐状态概率向量：T<k> = N * T<k - 1>,
				 	2) 第k次时可见状态概率向量：U<k> = T<k> * U<k - 1> = T<K> * U
					---并且有推论4：
					因为已经估计出了隐马尔科夫链模型（参数矩阵，各个），所以可以用来对每次的输出序列估计最可能的隐状态向量（最大似然法）。
				>拓展：更进一步，希望知道状态改变的在时间上的体现，因为改变总是需要时间的，如果可见状态值在连续区间内的任意值都可以，而且改变不会突然进行--比如观察速度--这种状态，这种状态的改变是连续的，且正好与上一时刻的加速度相关，所以加速度可以看做作是隐状态，而速度看作是可见状态（而且这种状态改变是一直向下一个状态靠近，是一种有规律的状态变化，是一种加速度一旦明确那么下一时刻就是百分之百概率知道速度是多少，即转换概率是1，或者说是一种路程是可测状态，速度是隐状态，那么路程状态在不断转换，且有一个状态转换速率，可以决定下一个可见状态的状态--值），
				>定义：连续时间上产生的一系列可见状态值：它的隐状态即决定它下一时刻可见状态值的这一时刻的度量：（可见）状态转换速率（定义为可见状态值的导数，在产生连续状态序列方向上的变化率）
					而这个不断转换的状态叫做状态函数。

	>问题：如何产生任何一种分布的随机序列？比如高斯分布的随机序列？
	       即已知分布函数f，求一个序列s。首先，最简单的，均匀随机序列，比如[1,5]之间的均匀随机分布序列。
		----或许通过均匀分布随机序列的计算，可以产生任何一种分布的随机序列。事实上既可以：计算概率积分：因为这个可以考虑到概率值进来，而且是单调函数。
		----法2：如果用两个维度上的均匀随机序列的方式：x-y两个轴上都是均匀的概率分布序列，那么此时整个平面上每个点都是等概率的，在分布函数内自然也是，这样，完全看分布函数某处的值，如果这个值越大，显然它上的点就“相对越多”--或者说取值到的概率就越大，这样就刚好满足概率分布函数的定义。因此这样二维分布序列取分布函数内的点，所得到的序列其实就是这个分布函数的序列!!!!
