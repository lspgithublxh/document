   	>马尔科夫过程：
		     假设有一个集合E,其中元素都是整数，如果有一个随机序列，序列中每个元素从E中取，而且随机序列在构造的过程中有：第i个元素出现为什么元素与前面出现过的全部i - 1个元素是什么都没有关系。
		     推论1：随机序列中的元素可以重复。
		     推论2：随机序列中第m + n个元素出现为j的概率，等于前面n个元素为任意元素时第m + n个元素出现j的概率。即第n个元素出现什么，会影响到底n + m个元素出现什么，但是不会影响到之间的m - 1个元素会出现什么。。这是一种间隔影响、跳跃影响。
		     推论3：如果上述中与m无关，即m为其中1步到n步中的任何一步，这是齐次马尔科夫链，那么从元素i到元素j的概率p<ij>有全部种可能，与中间到达某个元素的概率，和再从这个元素到j的概率都有关系，且（某个元素也是任意的）。且i是什么，j又是什么，是什么都可以。
			    而这样p<ij>就可以形成一个矩阵，叫做转移矩阵。对于元素0-M的正整数序列为集合的例子，如果每次转移只能是相近的元素，即转移到相邻的整数，那么这是一个一步转移矩阵。

		     >隐马尔科夫链：
		     		>转换概率：开始到了一个状态，然后从这个状态直接的不经过其他概率性操作动作而概率性的到达下一个状态，这个概率就是转移概率。显然，下一个状态有多少个，这个转移概率就有多少个。且这个状态是隐含状态。
				>输出概率：到达一个隐含状态后，再让这个状态所代表的具体事物产生一个具体的输出，而输出是多种的，每种有一个概率，成输出的每种是一个输出状态、可见状态，每种的概率是输出概率。
			 	 推论1：构造一个隐马尔科夫链：
					已知一个起始隐状态，已知各个隐状态之间的转换概率，又已知每个隐状态输出各个状态的概率（即输出概率）,那么自然可以产生一个隐马尔科夫链----即这些信息足以产生一个马尔科夫链。（充分）
				 推论2：已知序列，各个隐式概率和输出概率，求最可能的隐马尔科夫链。
					1) 那么从最后到最初：开始：转换概率 * 输出概率 中最大的那组作为最后两个隐马尔可夫链的节点。再不断向前推。
				        2) 因为开始状态确定后，后面就可以确定了，所以可以分别计算开始状态为全部可能的隐状态，然后依次往下计算下一个节点出现某某的概率，这样推到结尾，得到隐状态数目个概率链，求乘积中最大的那个就是最可能的了。
				 常见的问题：输出的序列是容易观察到的，但是隐状态是难以观察到的，所以想通过易知推测难知。
					     此时还需要：转换概率和输出概率。
				 -------------这就是常见的问题模型。且：输入概率阵、转换概率阵、输出概率阵，都是明确的。

				 推论3：已知序列k个，每个序列长l，求最开始的状态概率向量(N,1 * n)和隐状态转移概率矩阵(T,n * n)和可见状态输出矩阵(U,n * m)。
					有k个，可以推知这个序列各个位置各个可见状态的概率，共：m * l个概率。利用这个可以推知（理论上）隐马尔科夫链的上述三大参数矩阵。（条件之一：l >= ( n + n * n + n * m) / 3）。
					---因为有以下规律：
					1) 第k次时隐状态概率向量：T<k> = N * T<k - 1>,
				 	2) 第k次时可见状态概率向量：U<k> = T<k> * U<k - 1> = T<K> * U
					---并且有推论4：
					因为已经估计出了隐马尔科夫链模型（参数矩阵，各个），所以可以用来对每次的输出序列估计最可能的隐状态向量（最大似然法）。

	>问题：如何产生任何一种分布的随机序列？比如高斯分布的随机序列？
	       即已知分布函数f，求一个序列s。首先，最简单的，均匀随机序列，比如[1,5]之间的均匀随机分布序列。
		----或许通过均匀分布随机序列的计算，可以产生任何一种分布的随机序列。事实上既可以：计算概率积分：因为这个可以考虑到概率值进来，而且是单调函数。
		----法2：如果用两个维度上的均匀随机序列的方式：x-y两个轴上都是均匀的概率分布序列，那么此时整个平面上每个点都是等概率的，在分布函数内自然也是，这样，完全看分布函数某处的值，如果这个值越大，显然它上的点就“相对越多”--或者说取值到的概率就越大，这样就刚好满足概率分布函数的定义。因此这样二维分布序列取分布函数内的点，所得到的序列其实就是这个分布函数的序列!!!!

						对于第二个问题，有区间放缩法：比如X是[0, 1]之间的均匀分布，（连续分布），那么对于高斯分布的Y，其取值区间是正负无穷之间，所以要将[0,1]之间的数映射到正负无穷之间，容易想到一种类似反正切函数的函数，而且这种放缩可以是不均匀的，即[0,1]之间的部分取值小区间映射到正负无穷区间上很宽的子区间，而同样的另一个小区间可以映射到很窄的子区间，这个应当与目标随机变量的分布函数有关系，而且应当是由这个分布函数决定,而这个[0,1]之间的映射---即目标分布函数的自变量的在正负无穷上的每个取值，都要对应[0,1]上的某个数，而且不重复，而且放缩比例和取值概率对应---缩小比例大取值区间上的概率当小，综合那么求概率的面积积分是可以的，满足条件要求的。
						对于第一个问题：可以等效转化含义：从自变量的慢慢变化，可以看到f(x)的值的变化，对于单调函数，假想自变量等分为n个小区间，在某一个小区间上，自变量的取值概率是一样的，取每个小区间的概率是一样的，而小区间对应的因变量也有一个区间，这个区间的长度和自变量的区间长度不同，长度可能放缩了，而且因为自变量的均匀取值，那么也是均匀放缩，导致概率也同等地被稀释了放缩了那么多，显然，放大了区间则概率缩小，反之亦然，而这个区间长度之比就是概率之比的反比，而区间长度之比就是函数f的导数的绝对值（区间足够小），因此就得到了f(X)的概率分布：|f<'>(x)<-1>|。

	>马尔科夫性：
		   >定义：一个随机变量在时间上不断的取值，在t = m时，取值为i,是有概率的，当实际上如果取到这个值之后，它可以确定t = m + n处的取各个值的概率分布,比如取j的概率,因为 t = m说明那时t = m 之前的取值已经确定了，取了哪些值并不清楚，即有多种可能（多种可能的序列）,但是只要 t = m处X取值为i，无论之前m - 1 次取值了什么样的序列组合，那么t = m + n出的分布函数就只会是一个确定的不变的分布，这就是随机变量X的马尔科夫性（确定现在，能够确定未来（概率分布），但不能记住过去--即不能确定过去（过去的每一处的概率分布在这时此时都是不清楚的）（下一步的走，只根据当前的值，不根据当前之前的值））。数学描述即：P(Xm+n = j | Xm = i) = P(Xm+n=j | Xm=i, Xm-1 = i2, ...X1 = im)	，须知P(Xm+n = j)这个概率是和之前的概率不相同的---因为这个概率要从头开始一步一步算，而之前的概率只需要从m处开始算就可以。
			  而i是可以各种各样的值，（在取值区间内）,当具体取不同值时，m+n处的取值概率分布是不同的。所以要计算m+n出的取值概率分布，只需要从m处的取值分布开始算就可以（每个取值都能确定m+n处的分布，所以这些分布合起来，就能确定m+n处的分布，即已知m处的分布，而确定m+n处的分布....推广开来，如果m处取值已经确定，那么可以推出m+n处的取值分布，而m+n处的取值分布又能确定m+n+n处的取值分布，...），如果m处已经确定了什么，那么直接从这里开始算就可以了，不用从头开始算。
			  而n>=0都可以。如果n只能是一个确定的值，比如2，即m处的值不能确定m+1处的分布（这个分布和m处之前的取值有关系，之前取值不同而分布不同），仅仅能确定m+2处的分布，同理后推。这也是一种马尔科夫性，这种性质的现实过程现实随机变量是存在的。
			  而如果对于n>=0的任意取值，都可以，都有马尔科夫性影响，即m处的取值,能够确定m处以后的概率分布,即，以后各处具体取什么值，构成一个一个的序列，统计这个序列每处的取值概率（分布），可以看到正符合m处确定的未来的概率分布。而m+1处取什么值，是一个概率，而一旦确定取了什么值，那么m+2处的取值概率分布就是一定的，而且是m处确定的分布的一个分量。。相当于m处开始走一个路径，有多种选择，下一步也有多种选择，而且下一步的每种选择都确定了下下一步的每种选择及其概率，...，这样就延展为了一张树网。而未来某处取了某个具体值，而下一处的取值分布已经确定了，即下一步完全由这一步的取值而定，如果这一步的取值是一个区间，下一步的取值也是同样一个区间，显然，这张树网本质上就可以提出，个数为区间中值的个数的这么多个不同的分布函数（可以是无数个分布函数），这就是形成这个马尔科夫链的需要的全部知识（条件）,这个知识量化表示就是：F(t<m>,t<m+Δm>),令第一个参数为t,第二个参数为x,则其中t,x都属于某个区间E,且t作为参数（区间在E上的可变参数），x作为分布变量（区间在E上的分布变量）,整体含义就是m = t时候确定的，m+1处的概率分布。。对于离散取值，这是一个转移矩阵，阶数为取值的个数。
			  再如果m取值任何都可以，那么这是齐次马尔科夫链。
			  >具体的计算：
				    >离散取值：m处的概率分布向量X，则m+n处的概率分布向量Y,一步一步的从m处开始计算，可得:Y = X * N<n>，其中N是转移概率矩阵，m对于任意>=0的取值成立，n >= 0。
				    >连续取值：类似离散的累积计算方法，推知如果上一处的取值分布为f(y,x),那么下一处的取值分布就是∫<E>f(y,t) * f(t,x)dt。这个推导可以利用立体坐标系中f(y,x)的样子通过假设某时取值为y来推导（这实际上是一个递推公式）。
